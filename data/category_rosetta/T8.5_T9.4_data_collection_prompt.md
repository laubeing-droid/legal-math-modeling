# 任务一：T8.5 补充 US/HK 视角 Claim Mapping（~1,800 条）

## 已有数据说明

我们已从中国最高人民法院 20 卷实务全书中提取了 216 条跨法域案例，全部以 CN 法律主张为主。现在需要补充：

1. **US 视角的 CN-US 映射**（~800 条）：从美国判例法角度描述法律主张，再标注对应的 CN 法律主张
2. **HK 视角的 CN-HK 映射**（~400 条）：从香港判例法角度描述法律主张，再标注对应的 CN 法律主张
3. **US-HK 直接映射**（~300 条）：US 与 HK 之间的法律主张对应
4. **三法域完整映射**（~300 条）：CN/US/HK 三法域均有独立主张的案例

## 数据格式

```
pattern_id,domain,fact_summary,cn_claim,us_claim,hk_claim,mapping_status,hard_case,positive_control,notes
```

| 字段 | 说明 |
|------|------|
| `pattern_id` | 格式：`FP-{视角}-{NNN}`，如 `FP-US-0001`、`FP-HK-0001`、`FP-USHK-0001` |
| `domain` | 法律领域（contract/tort/criminal/admin/ip/labor/corporate/family/tax/procedure） |
| `fact_summary` | 中英双语事实摘要（≤200字） |
| `cn_claim` | 中国法下的法律主张（中文），如无对应填 `DATA_UNAVAILABLE` |
| `us_claim` | 美国法下的法律主张（英文），必须引用真实法条/判例 |
| `hk_claim` | 香港法下的法律主张（英文），必须引用真实法条/判例 |
| `mapping_status` | 见下方映射状态 |
| `hard_case` | `True`/`False`，边界争议案件比例 ≥30% |
| `positive_control` | `True`/`False`，已知答案对照组比例 ≥10% |
| `notes` | 法理差异说明（≤300字），必须解释三法域差异的法理基础 |

## 映射状态（补缺版）

| 状态 | 说明 | 补充量 |
|------|------|--------|
| `CN_US_PARTIAL` | CN↔US 部分映射 | ~500 条 |
| `CN_HK_PARTIAL` | CN↔HK 部分映射 | ~200 条 |
| `US_HK_PARTIAL` | US↔HK 直接映射 | ~300 条 |
| `TRI_JURISDICTION_PARTIAL` | 三法域均有对应但不完全等价 | ~300 条 |
| `TRI_JURISDICTION_MAPPED` | 三法域高度对应 | ~200 条 |
| `COLLISION` | 主张直接冲突 | ~200 条 |
| `ASYMMETRY` | 单向映射 | ~100 条 |

# 任务二：T9.4 补充 US/HK 赔偿金额数据（~300 条）

## 已有数据说明

已提取 663 条 CN 赔偿数据（187 条含金额），54 条 US，24 条 HK。需要补充 US/HK 真实判例金额。

## 数据格式

```
case_id,jurisdiction,domain,damage_type,initial_claim,final_award,iterations,court_level,year,converged,convergence_gap,notes
```

| 字段 | 说明 |
|------|------|
| `case_id` | 格式：`BD-{US/HK}-{NNN}` |
| `jurisdiction` | `US` 或 `HK` |
| `domain` | 法律领域 |
| `damage_type` | compensatory / liquidated_damages / punitive / statutory / emotional_distress / lost_profits / attorney_fees / litigation_costs |
| `initial_claim` | 原告索赔金额（美元/港元） |
| `final_award` | 法院最终判决金额 |
| `iterations` | 审理轮次（1=一审终结，2=有上诉，3=再审） |
| `court_level` | trial / appeal / supreme |
| `year` | 判决年份（2010-2025） |
| `converged` | 最后两轮变化 <10% 为 True |
| `convergence_gap` | 最后两轮差额（同币种） |
| `notes` | 法律依据和计算方法（≤200字） |
