# T8.5 + T9.4 跨法域法律数据采集提示词

## 文件定位

本文件是跨法域法律数据的采集提示词模板，用于指导 AI 系统批量生成法律映射数据。

## 已提取数据

来自最高法 20 卷全文数据库 8,712 页：

- **T8.5**：216 条 CN 视角跨法域映射（CN_US_PARTIAL 157 + TRI 25 + CN_HK 24 + CN_ONLY 10）
- **T9.4**：663 条赔偿数据（CN 585 + US 54 + HK 24），187 条含金额

## 与形式化的关系

采集的数据用于支持以下 Lean 形式化结构的实例化：

- `FiniteRosetta.lean`：跨法域映射的有限实例
- `FiniteGaloisAdjunction.lean`：语义格之间的伴随关系
- `TemporalKripke.lean`：时序法律推理的 Kripke 模型

## 数据格式

### T8.5 映射格式

```
pattern_id,domain,fact_summary,cn_claim,us_claim,hk_claim,mapping_status,hard_case,positive_control,notes
```

### T9.4 赔偿格式

```
case_id,jurisdiction,domain,damage_type,initial_claim,final_award,iterations,court_level,year,converged,convergence_gap,notes
```

## 映射状态

| 状态 | 说明 |
|------|------|
| CN_US_PARTIAL | CN-US 部分映射 |
| CN_HK_PARTIAL | CN-HK 部分映射 |
| US_HK_PARTIAL | US-HK 直接映射 |
| TRI_JURISDICTION_PARTIAL | 三法域均有对应但不完全等价 |
| TRI_JURISDICTION_MAPPED | 三法域高度对应 |
| COLLISION | 主张直接冲突 |
| ASYMMETRY | 单向映射 |

## 输出要求

1. T8.5：CSV 格式，表头在第一行，每批 100 条，按 mapping_status 分组
2. T9.4：CSV 格式，表头在第一行，每批 50 条，按 jurisdiction 分组
3. us_claim/hk_claim 必须引用真实法条或判例名称
4. 不含代码块标记，直接输出纯 CSV 文本
