# History Archive Guide

`docs/history/` is an archive, not the current release boundary.

Use this directory for:

- phase summaries,
- earlier modeling assumptions,
- methodology notes,
- historical Socratic audit snapshots.

Do not use this directory as the source of current release claims.

## Current Truth Sources

For current status, read these instead:

1. `docs/formal-release/FORMAL_RELEASE_REPORT.md`
2. `docs/formal-release/FORBIDDEN_CLAIMS.md`
3. `docs/final-closure/final-report.md`

## Archive Files

- `development_log_20260523_0614.md`
  Early-phase modeling and audit archive.
- `development_log_20260616_0617.md`
  Repository-creation and first-release archive.
- `llm_machine_provability.md`
  Historical methodology note on machine-provability assumptions.
- `socratic_200_rounds.md`
  First 200-round Socratic audit summary archive.
- `socratic_200_rounds_deep.md`
  Second 200-round deep audit summary archive.

## Staleness Rule

Some historical files may contain:

- older theorem counts,
- earlier Banach assumptions,
- pre-release branch references,
- ambitions later downgraded by audit.

Those statements are preserved as historical context only.
