# 04 算法实现分析

日期：2026-06-28（原版 2026-06-11）

## 1. 目标

本文件分析 juris-calculus 项目中数学模型的算法实现，对应 59 个 Python 理论模块和 25 个 Lean 形式化文件。

## 2. Lean 形式化实现

### 2.1 Horn 闭包实现

Lean 文件：`HornDefinitions.lean`、`HornFixedPoint.lean`、`HornOperationalRefinement.lean`

算法流程：

```
Stage 1: 定义 Horn 子句和闭包算子 (HornDefinitions.lean)
Stage 2: 证明闭包算子单调 (FiniteMonotoneIteration.lean)
Stage 3: 证明有限域上不动点存在 (HornFixedPoint.lean)
Stage 4: 操作精化 (HornOperationalRefinement.lean)
```

### 2.2 Dung AAF 实现

Lean 文件：`DungDefinitions.lean`、`DungAAF.lean`、`DungFixedPoint.lean`

算法流程：

```
Stage 1: 定义论证、攻击、可接受性 (DungDefinitions.lean)
Stage 2: 定义特征函数 (DungAAF.lean)
Stage 3: 证明 grounded extension 作为不动点 (DungFixedPoint.lean)
```

### 2.3 Banach 不动点实现

Lean 文件：`BanachContraction.lean`、`BanachFixedPoint.lean`、`BanachComplete.lean`、`BanachWeightedNorm.lean`、`BanachEffectiveNodes.lean`、`WeightedSupNorm.lean`、`SupZeroLemma.lean`

算法流程：

```
Stage 1: 定义加权范数和完备性 (BanachWeightedNorm.lean, WeightedSupNorm.lean, SupZeroLemma.lean)
Stage 2: 定义压缩映射 (BanachContraction.lean)
Stage 3: 证明 Banach 不动点定理 (BanachFixedPoint.lean)
Stage 4: 完备空间性质 (BanachComplete.lean)
Stage 5: 有效节点分析和证书 (BanachEffectiveNodes.lean, BanachCertificate.lean)
```
