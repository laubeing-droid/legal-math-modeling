# 05 优化报告：从数学模型到工程提升

## 1. 优化原则

1. 先修证据边界，再修算法性能。
2. 先拆非单调逻辑，再谈不动点证明。
3. 先做数据质量门控，再谈统计校准。
4. 先保留 jurisdiction 差异，再谈统一抽象。
